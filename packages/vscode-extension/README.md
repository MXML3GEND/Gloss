# Gloss VS Code Extension

Developer ergonomics for Gloss:

- Hover translation keys inside `t("...")` / `translate("...")` to preview all locales.
- Run command `Gloss: Open Key In Browser` to jump straight into Gloss.

## Settings

- `gloss.serverUrl` (default: `http://localhost:5179`)

Make sure your Gloss server is running before using hover previews.
